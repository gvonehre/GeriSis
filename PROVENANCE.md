# Provenance

This repository is a static React + Vite frontend demo/preview build for evaluation.

## Intended production architecture

For production delivery, this frontend is intended to be integrated into an Astro shell for SEO and connected to a custom backend "brain".

## Backend "brain" responsibilities

The production backend layer is intended to handle:
- automated email flows;
- booking capture and orchestration;
- confirmation workflows;
- calendar integrations and synchronization;
- site editing workflow;
- CMS integration or a custom CMS implementation.

## Not included in this repository

This demo repository does **not** include:
- backend source code;
- secrets or environment credentials;
- production infrastructure provisioning/configuration;
- third-party service credentials.
